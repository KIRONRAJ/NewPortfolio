# Kironraj Portfolio

A secure, high-contrast, interactive cybersecurity-themed portfolio website.

## Core Features
- **Interactive Cyber Terminal**: A simulated terminal environment where users can type commands (e.g., `help`, `about`, `skills`, `projects`, `theme`, `hack`) with optional synthesized mechanical keyboard typing sounds.
- **Multiple Cyber Themes**: Instant color profile swap between Green, Dracula Purple, Retro Amber, and Stealth Monochrome (via theme toggle or CLI input).
- **Responsive Terminal Placement**: Structured design positioning the CLI window directly below the manifesto header for optimal reading hierarchy.
- **Persistent Preferences**: Saves theme selections and sound effects to local storage.

## Key Projects
- **Replay Attack Prevention in Smart Car IoT Systems** *(Ongoing Master's Project)*: Lightweight Hybrid Nonce-Counter framework developed to mitigate automotive IoT security gaps.
- **Intelligent Aquaponics**: Automated aquaculture and hydroponics visual tracking hub.
- **Wireless FM Transmitter Module**: Low-power hardware modulation and transmission system.

## Development & Build
1. Install base node dependencies:
   ```bash
   npm install
   ```
2. Launch the developer server:
   ```bash
   npm run dev
   ```
3. Compile production bundle:
   ```bash
   npm run build
   ```
